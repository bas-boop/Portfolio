Portfolio: Vormen Vs Formen

Description

Vormen Vs Formen is mijn tower defence, gemaakt in Unity. Dit was een school project die ik in mijn eentje heb gemaakt. Heb daarom ook veel verschillende dingen gemaakt.
Heb gewerkt aan een Tile systeem, placing van towers op de tiles, schietende towers, money syteem, een shop, enemy's die werken met een wave spawner, upgraden en selling van towers met world UI.
Wat ik zelf het trots op ben is dat er een element type systeem is. Bijvoorbeeld een fire enemy, die krijgt minder damage van een fire tower.

Aan het einde van de school periode moesten we een lijst maken van wat er mist in de game volgens jou.
Hier heb ik later ook aan gewerkt, waaronder een main menu, pause screen, verschillende level's. Wat wel nog mist is sound.
