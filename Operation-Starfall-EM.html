Description

De eerste paar dingen waar ik aan in heb gewerkt in dit project, is het tweaken van de player movement.
Waaronder de jump momentum, ledge grabbing en releasing en deceration.
Dit heeft mij geleerd hoe om al bestaande code te lezen.

Heb aan het zelfde project ook gewerkt aan Custom Physics System en Pickup System.

De git repository is op privé!
Het wordt een commerciële game.

Jump momentum (Version 1):

Before:

After:

Wanneer je tijdens het lopen zou springen en dan lopen los laat dan zal je naar benden vallen, alsof je een onzichtbare muur zal raken.
Beetje onlogische, ik heb er voor gezorgd dat je momentum zal blijven houden.
Dit heb ik gedaan met een force te geven wanneer je geen input meer geef.


Jump momentum (Version 2):

Bij de eerste versie verloor je wat controlle, want je krijgt een set distance mee geven, die je wel kan om draaien.
Alleen je kon geen tap geven en een beetje verder komen.

Daarom verander ik de air accleration dynamis met wel of niet input geven in de lucht.
Hierdoor kan je ook uit stilstaan graduweel bewegen in de lucht.


Release Ledge:

Optimaliseren van het loslaten van de ledge wanneer je er aan hangt.
Je kon er namelijk alleen boven op komen, dus we hebben het gemaakt dat je het gewoon kan loslaten, waardoor je naar beneden valt.
Dit geeft meer opties en logica aan het spel.



Decelerator:

Als je van een hoogte zal komen naar de grond, dan zou je momentum houden wanneer je land op de grond.
Hierdoor sta je niet ineens stil op de grond wanneer je heel snel ging.
Dit hebben we gedaan via de forces van de Force System.


Deceleration:

Heb ook gewerkt aan de Deceleration van de speler.
Dus als je aan het lopen bent en stopt met input geven, dan doe je nog een paar kleine stappen totdat je echt stil staat.
Hierdoor sta je niet gelijk stil alsof je een muur raakt.
